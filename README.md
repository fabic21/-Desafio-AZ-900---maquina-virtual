Sobre o Projeto
Este repositório documenta minha jornada no curso AZ-900, abordando os fundamentos da computação em nuvem com Microsoft Azure.

## Aulas Assistidas
Todas as vídeo-aulas foram assistidas conforme solicitado no desafio. Os principais temas abordados incluem:
- Conceitos de nuvem: IaaS, PaaS, SaaS
- Benefícios da nuvem: escalabilidade, economia, segurança
- Serviços principais do Azure: VMs, Blob Storage, App Services
- Segurança e governança: RBAC, Azure AD
- Gerenciamento de custos e precificação

## Experiência Prática
Realizei a criação de uma máquina virtual do Windows no Portal do Azure.

## Capturas de Tela
As imagens estão organizadas na pasta `/images`, incluindo:
- Tela de criação da VM
- Configuração de rede
- Conexão via RDP
